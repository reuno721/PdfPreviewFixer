# Pdf Preview Fixer (PPFixer)

Windows에서 PDF 파일의 미리보기가 차단되는 문제를 해결하는 유틸리티입니다.

> **"미리 보려는 파일이 컴퓨터에 위험할 수 있습니다. 파일과 파일을 받은 출처를 신뢰할 수 있는 경우 파일을 열어서 파일 내용을 보십시오"**  
> 특히 이 경고와 함께 탐색기에서 “PDF 미리보기 안됨” 등 미리보기가 막히는 문제 해결에 도움을 줍니다.

<br>

## 📥 다운로드

**[최신 버전 msi 설치 파일 다운로드 바로가기](https://github.com/reuno721/PdfPreviewFixer/releases/latest)**

- 최신 버전 msi 설치파일 `PdfPreviewFixer-Setup-v1.0.0.msi` 1개만 다운로드하세요.
- Source code 압축파일은 개발자용

<br>

## 🔧 설치 방법

1. 다운받은 `.msi` 파일 실행
2. Windows 보안 경고 '추가 정보' 보이면 → **"실행"** 클릭
3. 관리자 권한(UAC) 요청 시 **"예"** 선택
4. 설치 완료

<br>

## 🚀 사용 방법

### 개별 파일 처리
1. PDF 파일 선택 (다중 선택 가능)
2. 마우스 오른쪽 클릭
3. **"PDF 차단 해제 (선택 파일)"** 클릭

### 폴더 전체 처리
1. 폴더 빈 공간에서 오른쪽 클릭
2. **"PDF 차단 해제 (현재 폴더)"** 클릭

<br>

## 💡 단계별 해결 프로세스
```
처리 완료 → "미리보기가 보이나요?"

✅ [네, 보입니다]
   → 종료

❌ [아니요, 아직 안 보입니다]
   → 추가 조치 안내
   
   1단계: F5 새로고침 / 탐색기 창 재실행
   2단계: 탐색기 강제 재시작 (경고 및 확인 후 실행)
          → 대부분 경우 해결 가능
```

<br>

## ⚡ 주요 기능

- ✅ **Zone.Identifier 제거** (인터넷에서 다운로드된 파일 차단 해제)
- ✅ **미리보기/썸네일 갱신을 위한 정리 및 안내** (윈도우 탐색기 미리보기 갱신)
- ✅ **다중 파일 선택 지원** (한 번에 여러 파일 처리)
- ✅ **한글/특수문자 파일명 지원** (공백 포함 파일명도 정상 작동)
- ✅ **단계별 안내** (사용자가 선택하는 해결 방식)

<br>

## ⚠️ 제한 사항

Windows 탐색기 미리보기 시스템은 OS 내부 동작(블랙박스)이 포함되어 있습니다.

- 일부 환경에서는 탐색기 재시작이 필요할 수 있습니다
- 네트워크/공유 폴더 및 OneDrive 등 클라우드 폴더에서는 반영이 제한/지연될 수 있습니다
