# Pdf Preview Fixer (PPFixer)

Windows에서 PDF 파일이 아래와 같이 차단됨 상태로 유지되어 탐색기 미리보기가 작동하지 않는 문제를 해결하기 위한 유틸리티입니다.

> "미리 보려는 파일이 컴퓨터에 위험할 수 있습니다. 파일과 파일을 받은 출처를 신뢰할 수 있는 경우 파일을 열어서 파일 내용을 보십시오"

<br>

## 다운로드
상단 Releases에서 최신 *.msi 설치파일 1개만 다운로드하세요.

<br>

## 설치 방법
1) 다운받은 MSI 설치파일 실행
2) 설치 중 Windows 보안 경고가 나올 수 있습니다.
3) 권한 요청(UAC) 나오면 '예(Yes)' 선택
4) 설치 완료

<br>

## 실제 사용 방법
1) 미리보기 차단된 폴더로 이동
2) 개별 파일 선택 또는 빈 공간에서 마우스 오른쪽 클릭
3) 'PDF 차단 해제' 실행
4) 결과에 따라 추가 작업 진행

<br>

※ 사용 참고사항 
- 단계별 차단 해제로 “미리보기가 보이나요?”
- Yes 선택 : 프로그램 종료
- No 선택 1단계 : F5 새로고침 / 탐색기 창 닫고 다시 열기
- No 선택 2단계 : 탐색기 재시작 선택 (경고, 확인 후 실행)

<br>

## 한계사항 고지
윈도우 탐색기(Windows Explorer) 미리보기 캐시는 OS 동작(블랙박스) 영역이 있습니다.
때문에 100% 자동 차단해제의 보장은 어려울 수 있습니다. 
다만 단계별 해결 루틴으로 대부분의 PDF 파일에 대한 미리보기 차단 해제가 가능합니다.
